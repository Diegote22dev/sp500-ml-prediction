# 📊 SP500 ML Prediction

## 📌 Descripción

Este proyecto tiene como objetivo predecir si el índice S&P 500 va a subir o bajar utilizando técnicas de Machine Learning.

## 🎯 Objetivo

Desarrollar un modelo de clasificación binaria que permita anticipar la dirección del mercado (sube o baja).

## 📂 Dataset

Se utiliza un dataset de datos históricos del S&P 500 en frecuencia horaria (H1), que incluye variables como precio Bid/Ask y volumen.

## 🧠 Enfoque del modelo

Se abordará el problema como una tarea de clasificación supervisada, utilizando modelos como:

* Logistic Regression
* Random Forest
